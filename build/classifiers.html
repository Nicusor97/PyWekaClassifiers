predict(query_data, verbose=False, distribution=False, cleanup=True)[source]¶

Iterates over the predicted values and probability (if supported). Each iteration yields a tuple of the form (prediction, probability).

If the file is a test file (i.e. contains no query variables), then the tuple will be of the form (prediction, actual).

See http://weka.wikispaces.com/Making+predictions for further explanation on interpreting Weka prediction output.
